Attestation generator

The french government declared a second national lockdown that has been going on since November 1st because of the coronavirus. Apparently the infection rates and death cases were too high. This wasn't my experience (I haven't heard of anyone suffering from the virus) and as everyone knows, anectodal evidence is always a good basis for judgement.

This lockdown is way more permissive than the first one in spring. You can do basically anything you want, as long as you have a paper where you allow yourself to do so. This is the so-called "attestation".

I can go outside and buy food.

Signed: me

You could print the document, but it would be a waste of paper to print a new one every day. Fortunately, you can generate a pdf by using a government website. It is allowed to use the pdf file on your phone instead of having a printed version.

Through some weird turn of fate, the code of the mentionned website was made public with a MIT licence: github link.

This was the occasion to make myself a tool to automatically and quickly generate the pdf when I need it. It would need to work on a phone and offline, and the easiest solution I found was making a static HTML file that you open with the web browser on your phone.

When using a local HTML file, Firefox would refuse to open other local files because of "security" reasons. It was maybe possible by changing some setting, but the easier and funnier way was to put everything in a single file.

You can find the result here: github link. A number of people have told me that this tool is completely useless since the government website actually works very well, but it had to be done, if only for the principle of it.

The development process was copy/pasting javascript code until it worked. Webdevs sure have it easy.
