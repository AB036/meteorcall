Portal VVVVVV

The original game

VVVVVV is a 2D platformer released in 2010 by Terry Cavanagh. It's a fun game with simple but engaging mechanics and the soundtrack is great.

The source code of the game was released in January 2020 (Announcement, GitHub repo). It's not free software, but the license allows anyone to compile the game and run it. The assets are proprietary though, but they can be downloaded here.

If you have never played this game, I definitely recommend it. You just need to compile it yourself, which should be easy for you.

Some quick instructions for a debian-like OS that might help you:

sudo apt install libsdl2-dev libsdl2-mixer-dev
git clone https://github.com/TerryCavanagh/vvvvvv
cd VVVVVV/desktop_version
mkdir build
cd build
cmake ..
make
wget https://thelettervsixtim.es/makeandplay/data.zip
./VVVVVV

(More instructions in the GitHub repository)

VVVVVV portal mod

Since the code was public, I decided to look inside, and eventually to make a small mod (link).

I added portal mechanics using the mouse. Left and right click create blue and orange portals, and the player can teleport between the two.

Inverting gravity is still possible. When you combine it with the portals, it made the game too easy. To keep the game interesting, I made small modifications on a lot of levels.

If you have better ideas about level design, don't hesitate to send a pull request. The levels are directly in the source code.

My first reaction after looking at the original code was criticism. It isn't """clean code""". After working on this project for some time, I realized the code allows for quick iteration and that's invaluable when making a game. Also, I put half of the portal logic in a function called gamerender.

Taking inspiration from hardcoded levels, I hardcoded the portals sprites. This was actually useful because I didn't have to modify the building process to include a new asset.

I think there are some efforts to refactor the game and correct the numerous bugs. Good luck guys.

Closing words

It was a fun project that was only possible because VVVVVV exists and its code was released, so thanks Terry.

Should you wish to play my mod, the building process is the same as the original game. I hope you enjoy it.

By the way, "Piercing the Sky" is the best track of the OST. It's not included in the original levels for some reason.
